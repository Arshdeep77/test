## Setup Instructions

Follow these steps to set up and run the project:

1. **Create a Virtual Environment:**

    ```bash
    python3 -m venv venv
    ```

2. **Install Dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

3. **Run the Development Server:**

    ```bash
    python3 manage.py runserver
    ```
